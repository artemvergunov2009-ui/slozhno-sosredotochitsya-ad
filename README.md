
# Проект: «Сложно сосредоточиться»

Учебный проект по созданию адаптивного лендинга с поддержкой темной и светлой цветовых схем. Основной упор сделан на современную верстку с использованием **CSS Grid**, **CSS Variables** и подхода **Mobile First**.

[Посмотреть макет в Figma](https://www.google.com/search?q=https://www.figma.com/design/AtRv4K5fwqfFZl9avwJuNW/3-%D1%81%D0%BF%D1%80%D0%B8%D0%BD%D1%82.-%D0%9F%D1%80%D0%BE%D0%B5%D0%BA%D1%82%D0%BD%D0%B0%D1%8F-%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%B0%3Fnode-id%3D0-1)

## 🎯 Цели проекта

  * Реализация адаптивности под три типа устройств: мобильные телефоны, планшеты и десктопы.
  * Применение методологии **Mobile First**.
  * Создание системы переключения цветовых схем (День, Неон, Авто) с использованием `prefers-color-scheme`.
  * Работа с современными CSS-функциями: `clamp()` для типографики и логическими свойствами.

## 🛠 Технологии

  * **HTML5:** Семантическая разметка (`header`, `main`, `section`, `article`, `footer`).
  * **CSS3:**
      * **Grid Layout:** Для построения сеток и адаптивности.
      * **Custom Properties (Variables):** Для динамической смены тем.
      * **Flexbox:** Для элементов навигации.
      * **Адаптивная типографика:** Использование формул `clamp()`.
  * **JavaScript:** Скрипт для управления классами тем.

## 📁 Структура проекта

```text
├── fonts/          # Шрифты проекта
├── images/         # Изображения и иконки (Favicon)
├── scripts/
│   └── script.js   # Логика переключения тем

│   ├── globals.css   # Сброс стилей и базовые настройки

│   ├── style.css     # Основные стили и медиазапросы
│   ├── dark.css      # Специфичные стили для темной темы
│   └── light.css     # Специфичные стили для светлой темы
└── index.html        # Основная разметка страницы
```

## 🚀 Как запустить проект

1.  Склонируйте репозиторий:
    ```bash
    git clone https://github.com/твой-аккаунт/slozhno-sosredotochitsya-ad.git
    ```
2.  Перейдите в папку проекта:
    ```bash
    cd slozhno-sosredotochitsya-ad
    ```
3.  Откройте файл `index.html` в любом современном браузере.

## ✅ Особенности реализации

  * **Темы:** Основной темой выбрана "Неон". Поддерживается как ручное переключение, так и автоматический выбор на основе настроек ОС.
  * **Доступность:** Декоративные элементы (REC, угловые рамки) реализованы псевдоэлементами и скрыты от вспомогательных технологий через `aria-hidden`.
  * **Производительность:** Реализована ленивая загрузка изображений (`loading="lazy"`).
