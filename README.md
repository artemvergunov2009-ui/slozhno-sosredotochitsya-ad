Проект: «Сложно сосредоточиться»
Учебный проект по созданию адаптивного многостраничного (одностраничного в рамках задания) лендинга с поддержкой темной и светлой цветовых схем. Основной упор сделан на современную верстку с использованием CSS Grid, CSS Variables и подхода Mobile First.


🎯 Цели проекта
Реализация адаптивности под три типа устройств: мобильные телефоны, планшеты и десктопы.

Применение методологии Mobile First.

Создание системы переключения цветовых схем (День, Неон, Авто) с использованием prefers-color-scheme.

Работа с современными CSS-функциями: clamp() для типографики и логическими свойствами (например, margin-block-start).

🛠 Технологии
HTML5: Семантическая разметка (header, main, section, article, footer).

CSS3:

Grid Layout: Для построения сложных сеток.

Custom Properties (Variables): Для легкого управления цветами и размерами.

Flexbox: Для выравнивания элементов в меню и карточках.

Адаптивная типографика: Использование формул clamp().

JavaScript: Готовый скрипт для динамического переключения классов тем (theme-dark, theme-light).

✅ Особенности реализации
Темы: Основной темой выбрана "Неон" (Dark). Светлая тема подключается автоматически через системные настройки или вручную кнопкой.

REC: Декоративный элемент в шапке реализован с использованием псевдоэлементов и скрыт от скринридеров через aria-hidden.

Производительность: Для изображений в галерее настроена ленивая загрузка (loading="lazy").

Фиксация: Фоновое изображение закреплено (background-attachment: fixed), что создает эффект глубины при скролле.
